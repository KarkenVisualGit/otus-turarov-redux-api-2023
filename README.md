### Развернуть проект

1.Клонировать репозиторий

```shell

gh repo clone KarkenVisualGit/otus-turarov-redux-api-2023

```

2.Установить зависимости

```shell

npm install

```

3.Запустить локальный сервер

```shell

npm run start
```
